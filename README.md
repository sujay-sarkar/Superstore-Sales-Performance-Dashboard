Superstore Sales Performance Dashboard | Power BI

Project Overview

    This project is an interactive Sales Analytics Dashboard built in Power BI to analyze sales performance, profit, and order trends across time, regions, categories, and customer segments. The dashboard is designed for quick analysis and easy decision-making.

Key Metrics

    - Total Sales: $2.26M
    
    - Total Profit: $279.09K
    
    - Total Orders: 9,785

Filters (Slicers) Used

    The dashboard includes the following interactive filters:
    
    - Year: 2018 – 2021
    
    - Month: January – December
    
    - Category: Furniture, Office Supplies, Technology
    
    - Region: Central, East, South, West
    
    All visuals and KPIs update dynamically based on the selected filters.

Insights Covered

    - Region-wise sales comparison
    
    - Category-wise sales distribution
    
    - Top 5 sub-categories by sales
    
    - Top states by sales and profit
    
    - Month-on-month sales and profit trends
    
    - Segment and category-wise sales performance

Tools Used

    - Power BI
    
    - Power Query (data cleaning and transformation)
    
    - DAX (measures for KPIs)

How to Use

    Open the .pbix file in Power BI Desktop
    
    Use the slicers to explore sales and profit insights
